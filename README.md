##  Exercise Sections

The exercises are organized into clear categories:

### **Section 1 — Conditional Logic**
> Core decision-making patterns used in conditional rendering.

### **Section 2 — Array Manipulation**
> Practice with map, filter, reduce, and transformations used for dynamic UI rendering.

### **Section 3 — String & Input Handling**
> Text formatting, form validation, and data cleaning.

### **Section 4 — State-like Transformations**
> Pure functions that simulate typical `useState()` updates.
